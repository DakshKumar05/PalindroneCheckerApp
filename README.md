# PalindroneCheckerApp# PalindromeCheckerApp

## Project Overview
PalindromeCheckerApp is a Java application that demonstrates the basic structure of a palindrome checking system. The project is organized using use cases, starting with application initialization.

## Objective
- Understand Java program structure
- Implement a clear execution flow
- Prepare the system for palindrome checking functionality

## Running Procedure
Compile the program:
javac UseCase1PalindromeCheckerApp.java

Run the program:
java UseCase1PalindromeCheckerApp

## Project Flow
1. Execution starts from the main() method.
2. A welcome message is displayed.
3. The application version is shown.
4. The system confirms initialization.

## Use Cases
Use Case 1: Application Entry and Welcome Message
- Serves as the entry point of the application.
- Displays system title.
- Shows version information.
- Establishes the startup process.

File: UseCase1PalindromeCheckerApp.java
